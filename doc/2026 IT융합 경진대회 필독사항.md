##### 기본 사항

회로 30점, 코드 70점 (각 모드마다 점수 기준 있음)



##### 대회시작

1\. 오캠 먼저 촬영 스톱워치 작동

2\. 응시번호로 바탕화면에 폴더 생성

3\. 프로테우스 실행 후 도면작성하고 저항 바꾸고

4\. 부저 내부 전압(오퍼레이팅 볼테이지)만 1V해야함 : 무조건 액티브로 사용해야함 다른건 안됨

5\. 모든 과정 끝나면 응시번호 파일에 PDF, hex, main.ino, 오캠 녹화파일(필요한 것 만 넣어서 제출)

6\. 손 들고 평가 시작(평가 시작하면 복사 붙여넣기를 제외한 모든 행동이 금지됨)



##### 파일 제출 단계

\-개인 스톱워치로 시간 측정

\- 감독관이 제시하는 종이에 완성 시간 기입 및 USB에 응시번호 파일 제출

PDF파일 출력 방법:

1\. 단축키 G로 그리드 없앤 후

2\. File -> Print Design : Fit To Page, Print In Colour 체크(줌 아웃하고 뽑기)

3\. 저장위치: 응시번호 폴더

code파일 출력 방법:

1\. Project -> Project Settings -> Embed Files 체크 해제

2\. 저장 위치에 Arduino Mega 1 파일 생성

3\. 그 안에 main.ino 파일과 debug.hex or debug.elf 파일 제출(되도록 hex파일 제출)

##### 주의사항

1\. Arduino Mega 저항 네모칸이 아닌 지그재그 모양

2\. 필요없는 부품은 배치하지 않음(감점 요인)

3\. 보기 쉽도록 노드 크기 키울 것

4\. WS2812 핀 확인 제대로 해서 연결 할 것

(부품 핀에 마우스 올려 놓으면 핀 네임 뜸)

5 .헤더(이름택) 글자크기가 서로 다르기만 하면 됨

6\. 선 배치: 꺾여있거나 겹쳐있으면 감점

7\.  FND부품과 LCD부품이 깜빡거리거나 안 뜨면 감점

8\. 전원 부분에 +5V인가 하지 않을 시 감점

9\. FND부품 저항 필요 없음

10.BUZZER 부품은 무조건 Active  형태로 사용할 것
